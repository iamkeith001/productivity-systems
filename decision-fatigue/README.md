# 決策疲勞管理

Decision Fatigue Management

把「太多選擇導致心智耗損」轉成可管理的系統：先盤點高頻耗損點，再建立固定規則、設定決策門檻、保留高能量時段給重要判斷，並用重要決策日誌累積可重用的判斷依據。

## 功能

- 耗損盤點：記錄高頻、複雜、帶壓力的決策情境
- 規則設計：把重複判斷改成固定化、門檻規則、決策框架或授權處理
- 分層管理：把小事自動化、中事標準化、大事深度化
- 重要決策日誌：保存背景、假設與最後判斷
- 本機資料儲存：localStorage
- JSON 備份 / 還原

## 使用

開啟 `index.html` 即可使用，不需要後端。
