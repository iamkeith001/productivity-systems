# 知識原子化 · Knowledge Atomization

把複雜知識拆解成小而清楚、可獨立理解的最小單位，提升學習與應用效率。精神源自 Zettelkasten（Luhmann 卡片盒筆記法）與 Obsidian／Roam 等第二大腦工具。

**立即使用：https://iamkeith001.github.io/productivity-systems/knowledge-atom/**

## 功能

- **原子**：萃取單一概念的知識原子——核心概念（一句話）＋用自己的話重寫（含「我理解的意思是／這可以用在／它提醒我」句型鷹架）＋標籤。搜尋、依標籤篩選；每個原子可與其他原子建立**雙向連結**，未連結的顯示為「知識孤島」。
- **圖譜**：知識圖譜視覺化，原子為節點、連結為線；點節點聚焦看內容與鄰居；統計原子數、連結數、孤島數。
- **重組**：選多個原子＋挑格式（短文／教學圖卡／簡報架構／研究假設），一鍵組成可編輯草稿，複製或存入重組紀錄。
- **學習**：四個特徵、五步驟、應用案例與專業解讀。

## 與知識蒸餾的關係

「知識蒸餾」把一批素材煉成一張精華卡（收斂）；「知識原子化」把知識拆成許多可連結、可重組的原子（發散再重組）。互補。
