# 情境預演法 · Premortem Thinking

把《情境預演法（Premortem Thinking）》變成可以在行動前使用的風險預演工具。

**立即使用：https://iamkeith001.github.io/premortem-thinking/**

單一 HTML 檔、零外部相依、離線可用、資料只存在你自己的裝置（瀏覽器 localStorage）。

---

## 核心理念

> 先替未來的失敗開一場檢討會，讓現在的行動更接近成功。

這個工具協助你完成一個 premortem 流程：

```text
定義目標 → 假設已經失敗 → 列出失敗原因 → 找出高機率高影響風險 → 轉成預防行動
```

## 四大分頁

| 分頁 | 對應步驟 | 功能 |
|---|---|---|
| **目標** | Define | 清楚寫下計畫、成功條件、時程與情境。 |
| **預演** | Imagine Failure | 假設計畫已經失敗，快速列出可能原因。 |
| **矩陣** | Prioritize | 以機率與影響評估風險，分成四個處理區。 |
| **行動** | Prevent | 把風險改寫成預防行動、負責人與追蹤項目。 |

## 怎麼用

### 電腦

直接用瀏覽器打開 `index.html` 即可。

### 開發預覽（本機伺服器）

```bash
python3 -m http.server 4180
# 然後開 http://localhost:4180/index.html
```

## 備份與還原

- 所有資料存在使用者自己的瀏覽器，不會上傳任何地方。
- 清除瀏覽器資料 = 紀錄消失，請在「行動 → 備份與還原」偶爾按「下載備份」。
- 換裝置時，用「讀取備份」還原。
