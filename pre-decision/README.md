# 預先決策法 · Pre-decision

在面臨選擇壓力前先做決策，減少臨場焦慮與決策疲勞，提高行動一致性。概念建立在決策疲勞、預先承諾與 Gollwitzer 執行意圖（如果…就…）的研究上。

**立即使用：https://iamkeith001.github.io/productivity-systems/pre-decision/**

## 功能

- **規則**：建立兩種預先決策——情境規則（如果…就…，會重複）與單一預先決定（某個時機的一次決定），含即時公式預覽、高頻場景標記與「降低選項／環境設計」欄位。
- **臨場**：情境真的發生時回來打卡——照規則做了 / 破功了。累積出遵循率與連續照做次數，看出哪些規則真的在幫你。
- **週檢**：每週回顧每條規則，保留有效的、停用太僵硬的；遵循率偏低會提示可能需要調整。
- **學習**：兩種形式、五步驟、應用案例與專業解讀。

## 與其他工具的關係

「如果…就…」與「行動觸發」的 If-Then 同源；預先決策直接降低「認知負荷」的外在負擔。可互相參照。
