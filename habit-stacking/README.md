# 微習慣堆疊法 · Habit Stacking

把新習慣綁定在既有行為上，降低養成難度，讓好習慣自然發生。概念源自 BJ Fogg 的 Tiny Habits 與 James Clear《原子習慣》的 Habit Stacking。

**立即使用：https://iamkeith001.github.io/productivity-systems/habit-stacking/**

## 功能

- **堆疊**：用公式「在我完成 A 之後，我會立刻做 B」建立堆疊——挑穩定的既有習慣 A 當觸發，掛上小到不抗拒的新習慣 B，可加環境設計與完成回饋。每組堆疊可「再疊一層」串成習慣鏈（視覺化為 A → B1 → B2）。
- **今日**：當天所有微習慣的打卡清單，完成度圓環；勾選時跳出你設定的小回饋。
- **進度**：每個微習慣的近 7 天打卡格、目前連續與最長連續天數，總覽微習慣數、今日完成率、最長連續。
- **學習**：公式與例子、五步驟、應用案例與專業解讀。

## 與其他工具的關係

堆疊決定「接在哪個舊習慣後面」；「兩分鐘啟動」幫你把 B 縮到最小、跨過開始門檻；「行動觸發」用 If-Then 與每週回顧優化單一觸發。三者可組合。
