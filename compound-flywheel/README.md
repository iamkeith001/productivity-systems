# 複利行動飛輪

Compound Action Flywheel

把「小而持續的行動」做成可追蹤的飛輪系統：先選一個值得長期累積的核心領域，定義每天最小行動，用每日推動、動能儀表板與每週回顧形成「行動 → 成果 → 信心 → 動機 → 行動」的循環。

## 功能

- 建立核心飛輪：選領域、最小行動、每日最小量、回饋指標與外溢成果
- 每日推動：快速填入最小量、記錄成果與阻力
- 動能儀表板：14 天累積、目前連續、最長連續、平均動能分
- 每週回顧：記錄有效成果、主要阻力與下週調整
- 本機資料儲存：localStorage
- JSON 備份 / 還原

## 使用

開啟 `index.html` 即可使用，不需要後端。
