# 能量區塊優化 · Energy Blocking

依照能量高低安排任務，讓表現最大化，避免在低能量時硬撐。從「時間管理」走向「狀態管理」，概念源自 Tony Schwartz 的能量管理與 chronotype 節律研究。

**立即使用：https://iamkeith001.github.io/productivity-systems/energy-blocking/**

## 功能

- **節律**：每天用 1–5 分記錄早上／中午／下午／晚上的精神狀態，系統推算你的「能量地圖」，替各時段標上能量等級（高／中／低／恢復）。
- **任務**：把待辦依能量需求分類（高：寫作研究決策／中：會議回信整理／低：歸檔排程／恢復：散步冥想），依分級篩選。
- **排程**：把任務排進對的時段。放錯會提醒——高需求任務排進低能量時段標「能量不足」，雜務占用高能量時段標「黃金時段可惜」。內建 60／75／90 分鐘專注衝刺計時器，結束自動提示恢復。
- **學習**：理論背景、四種能量區段、五方法、應用案例與專業解讀。

## 與能量儀表板的關係

「能量儀表板」（energy-first）幫你找出黃金時段；本工具把任務排進那些時段。兩者互補。
