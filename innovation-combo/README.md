# 創新組合思維 · Innovation by Combination

透過元素拆解與重新組合，創造全新價值。概念源自 Schumpeter 對創新的定義——「新組合」：將既有產品、技術、市場、資源或組織方式，以不同形式重新配置。智慧型手機＝電話＋相機＋網路＋定位＋應用平台。

**線上使用：https://iamkeith001.github.io/productivity-systems/innovation-combo/**

## 核心概念

創新的起點不必問「我能發明什麼全新的東西」，而是問「哪些已存在的元素，還沒被這樣組合過」。核心流程：元素拆解 → 跨域搜尋 → 重新組合 → 快速驗證 → 形成新價值。創新通常不是增加更多功能，而是改變元素之間的關係。

## 功能

- **課題**：先把問題定義清楚——不要「我們需要新系統」，要「如何降低新人學習時間？」（附句式範本）。
- **拆解**：現有元素依八面向拆開（功能／技術／對象／場景／通路／體驗／商業模式／資料），再向其他領域借原理建跨域元素庫（AI、自動化、遊戲化、視覺化、即時回饋、訂閱制、遠距協作）。
- **組合**：核心分頁——🎲 擲組合骰讓元素隨機相遇（組合矩陣的隨機抽格），強迫提出「如果 A＋B，會怎樣？」；先衝數量再用四面向評分（需求性／可行性／差異性／持續性），≥3 分標「值得原型」並解鎖最小原型欄位（先測哪個關鍵假設＋已測試打勾）。
- **回顧**：課題數、元素數（現有／跨域）、組合點子數、值得做原型數。

## 與同系列思維工具的差別

- **心智模型**（`mental-models/`）：收集思考框架（理解世界的透鏡）。
- **限制條件重構**（`constraint-reframing/`）：把限制翻轉成設計條件（單一問題的重新表述）。
- **創新組合思維**（本工具）：把既有元素重新連結出新價值（生成新方案的產線）。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`innovation_combo_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式

直接用瀏覽器開啟 `index.html`，或加到手機主畫面當 App 用。
