# 資訊飲食管理 · Information Diet

把資訊攝取當飲食管理：篩選、控制與優化資訊來源，減少過載，提升專注與判斷力。概念源自 Clay A. Johnson《The Information Diet》。

**立即使用：https://iamkeith001.github.io/productivity-systems/info-diet/**

## 功能

- **菜單（來源分級）**：盤點每個資訊入口，標記管道類型與營養分級（高／中／低／有害）。頂部「資訊營養標示」用色條呈現你的資訊飲食結構並給出判讀；可設定攝取時段作為提醒。
- **消化（輸入到輸出）**：每次攝取後選擇來源與產出類型（一句摘要／一個行動／一個判斷／一個問題／知識卡片／沒有產出），記錄輸出。統計產出率，換不到產出的來源標記「考慮降頻」。
- **週檢（每週清理）**：逐一檢視作用中的來源——「幫我做決策／只是讓我忙／該砍掉」，一鍵靜音或取消訂閱，記錄每週檢視。
- **學習**：理論背景、三標準、五方法、應用案例與專業解讀。

## 特性

- 單檔 HTML，純前端、離線可用
- 資料只存本機 localStorage，可下載／讀取 JSON 備份
- 深淺色主題、支援加入手機主畫面
