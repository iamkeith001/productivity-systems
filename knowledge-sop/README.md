# 知識複製 SOP · Knowledge Replication SOP

把個人經驗轉化為可複製的流程，讓團隊快速學會並穩定產出。概念源自 Nonaka & Takeuchi 的 SECI 模型：讓隱性知識與顯性知識彼此轉換，最後成為組織可持續使用的能力。

**線上使用：https://iamkeith001.github.io/productivity-systems/knowledge-sop/**

## 核心概念

把「我會做」變成「任何人看了都能做」。真正有價值的 SOP 不只是保存知識，而是降低新人學習成本、減少品質波動，讓團隊產出穩定放大——SOP 的目的不是讓人變成機器，而是讓基礎流程穩定，把精力留給更高階的判斷、創新與改善。

## 功能

- **清單**：三問挑高價值知識（最常被問？最容易出錯？最依賴資深人員？）——兩題以上「是」＝優先 SOP 化。
- **編寫**：五要素流程編輯器——輸入與產出、步驟清單、判斷點與升級條件、例外／常見錯誤／禁忌（真正的隱性知識）、檢核清單（做到標準 vs 只是做完）。附「標準化心法」提示：把模糊語句改成可執行語句。
- **驗證**：核心分頁——新人試行三檢驗（卡住→不夠清楚；結果不同→標準不明確；一直問人→例外沒寫完整），全過＝已驗證。加上版本管理：流程出錯／工具改版／規範調整／團隊回饋都記一筆並升版。
- **回顧**：SOP 總數、已驗證數、試行紀錄、版本更新次數。

## 與同系列知識工具的差別

- **知識壓縮力**（`knowledge-compression/`）：把複雜壓成可傳遞的一句話（表達層）。
- **學習金字塔重構**（`learning-pyramid/`）：管理自己怎麼學進來（輸入流程層）。
- **知識複製 SOP**（本工具）：把自己會的做出去、讓別人也會（輸出複製層）。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`knowledge_sop_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式

直接用瀏覽器開啟 `index.html`，或加到手機主畫面當 App 用。
