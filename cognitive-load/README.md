# 認知負荷管理 · Cognitive Load Management

把 John Sweller 的認知負荷理論做成可實際使用的工具：診斷 → 處方 → 追蹤。

**立即使用：https://iamkeith001.github.io/productivity-systems/cognitive-load/**

## 功能

- **診斷**：針對特定任務做 12 題快篩，量測三類負荷（內在／外在／增益，各 0–12 分）並產生判讀。
- **處方**：依負荷結構推薦五大方法（移除干擾、拆小步驟、視覺化、逐步增難、自動化）中的對應行動，勾選後存成減負計畫。
- **追蹤**：計畫卡追蹤行動完成率，可「再診斷」記錄第二次分數，顯示前後對比（內在／外在下降、增益上升）。
- **學習**：理論背景、三類負荷、五方法、應用案例與專業解讀。

## 特性

- 單檔 HTML，純前端、離線可用
- 資料只存本機 localStorage，可下載／讀取 JSON 備份
- 深淺色主題、支援加入手機主畫面
