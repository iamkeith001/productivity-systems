# 能量管理矩陣 · Energy Management Matrix

根據任務類型與個人狀態，安排每天最高效的工作組合。概念源自 Loehr & Schwartz《The Power of Full Engagement》：高效能的關鍵不是管理時間，而是管理能量。

**線上使用：https://iamkeith001.github.io/productivity-systems/energy-matrix/**

## 核心概念

不是問「我今天還有多少時間」，而是問「我現在適合做哪一種任務」。兩軸：任務能量需求（高／低）× 個人能量狀態（高／低），形成四種安排——高×高做深度工作、高×低別硬撐、低×高建未來資產、低×低做例行恢復。

## 功能

- **任務**：三問自動判定能量需求（需要高度專注？需要創造判斷？出錯成本高？）——兩題以上「是」＝高需求。
- **配對**：核心分頁——「⚡高能量／🌙低能量」一鍵切換當下狀態，即時顯示對應象限的任務清單：高能量給深度工作＋建資產清單；低能量給例行清單＋被卡住的高需求任務（⏸ 標示「等狀態回來再做」，不建議直接做）。完成就打勾。
- **節律**：連續記錄六時段能量（1–5），長條圖顯示各時段平均，自動標出**黃金時段**。
- **回顧**：任務高／低需求數、已完成、節律紀錄數、黃金時段。

## 與「能量區塊優化」的差別

- **能量區塊優化**（`energy-blocking/`）：週能量曲線＋任務排程對位＋專注衝刺計時器（規劃層）。
- **能量管理矩陣**（本工具）：當下配對器——「我現在適合做什麼」的即時決策（執行層）。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`energy_matrix_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式

直接用瀏覽器開啟 `index.html`，或加到手機主畫面當 App 用。
