# 學習金字塔重構 · Learning Pyramid Reimagined

用最有效的學習方法，讓知識真正內化並能應用。經典「學習金字塔」的留存率百分比（聽講 5%、教人 90%…）並無嚴謹實證，本工具重構它的核心精神：**學習效果不取決於接收多少資訊，而取決於是否主動處理、輸出、回饋與應用。**

**線上使用：https://iamkeith001.github.io/productivity-systems/learning-pyramid/**

## 核心概念

把學習重構成三層：**輸入層**（聽課、閱讀、看影片——最容易出現「我看懂了」的錯覺）→ **處理層**（提問、筆記、比較、摘要、討論——把外部資訊轉成自己的理解）→ **輸出層**（練習、實作、教別人、應用——真正讓知識內化的關鍵）。知識留下來要經過三次轉換：資訊→理解、理解→輸出、輸出→應用。

## 功能

- **主題**：建立學習主題時先定「輸出型」目標——解釋概念／解決問題／教給別人，目標句自動帶入第一步。
- **七步**：七步驟學習法逐步打勾＋每步一句紀錄——明確目標、主動預習（建立問題感）、深度理解（核心假設與失效情境）、練習與輸出、教導他人、應用到真實情境、反思與優化（三問回顧）。
- **三層**：核心分頁——每個學習行動記進輸入／處理／輸出層，三色長條顯示比例。零輸出或輸入佔比 ≥60% 時警示「看懂了的錯覺」；輸出佔比健康時給正向回饋。
- **回顧**：主題數、完成七步數、輸出行動數、輸出佔比。

## 與同系列知識工具的差別

- **知識蒸餾法**（`knowledge-distill/`）：把讀過的內容濃縮成本質（內容加工層）。
- **知識原子化**（`knowledge-atom/`）：把知識拆成可組合的最小單位（儲存結構層）。
- **學習金字塔重構**（本工具）：管理「怎麼學」的過程——確保每次學習走到輸出與應用（學習流程層）。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`learning_pyramid_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式

直接用瀏覽器開啟 `index.html`，或加到手機主畫面當 App 用。
