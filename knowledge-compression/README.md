# 知識壓縮力 · Knowledge Compression

真正高階的理解，不是知道更多，而是能把複雜變得可傳遞。概念連結 Chunking、Abstraction、心智模型與費曼技巧：真正厲害的人，往往不是記得最多，而是能快速抓到本質，並用更簡潔方式傳遞。

**線上使用：https://iamkeith001.github.io/productivity-systems/knowledge-compression/**

## 核心概念

真正的壓縮不是亂刪——保留核心、去除噪音、建立結構、強化可理解性，就像好的 ZIP 檔。知識壓縮力 ≠ 知道比較少，知識壓縮力 ＝ 能讓別人更快理解。

## 功能

- **壓縮**：壓縮工作台三步——① 找出真正重要的 20%（一行一個關鍵）② 記錄刪掉的噪音 ③ 濃縮成一句話高密度版本，選擇壓縮後結構（一句話／四欄表格／Checklist／流程圖／SOP／心智模型／FAQ）。
- **膠囊庫**：知識膠囊卡片——一句話以宋體引號呈現、核心關鍵列表、結構標籤；「複製膠囊」一鍵複製成可貼進簡報、筆記或 prompt 的格式。
- **檢驗**：費曼五項檢驗（1 分鐘講清楚／非專業者能懂／能轉圖表／能變 SOP／AI 能直接用），≥4/5 標「高壓縮品質」，未達標提示「回去再壓一輪」。
- **回顧**：膠囊數、高壓縮品質數、平均檢驗分、最常用結構。

## 與相近工具的差別

- **知識蒸餾法**：素材歸類 → 萃取洞見卡片 → 回流檢核。
- **知識原子化**：拆成最小概念單位＋雙向連結圖譜。
- **知識壓縮力**（本工具）：訓練「把一團複雜壓成一句可傳遞的話」的能力，並用費曼檢驗把關品質。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`knowledge_compression_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式
