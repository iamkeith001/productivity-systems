# 高階選擇權 · Strategic Optionality

好的人生系統，不只追求當下最佳，而是保留未來更多好選項。概念源自金融選擇權（Black–Scholes, 1973）：在不確定的未來中，「擁有選擇的權利」本身就有價值。

**線上使用：https://iamkeith001.github.io/productivity-systems/strategic-optionality/**

## 核心概念

高階選擇權是刻意保留、建立與擴大未來可能性的能力。它不是猶豫不決，而是有意識地保留未來自由度——當世界改變時，你還有多少可轉換空間。

## 功能

- **盤點**：選擇權資產盤點（可遷移能力／收入來源／平台管道／人脈圈／知識資產），每項打「可轉換度」1–5 分；頂部分散度格子即時顯示五類覆蓋狀況，缺口類別以紅框警示。
- **彈性**：五面向彈性自評（時間 buffer／現金流／學習空間／休息恢復／探索餘裕），儲存後顯示彈性儀表長條與歷次對比——彈性本身就是資產。
- **探索**：探索紀錄（跨領域閱讀、新工具測試、新產業接觸⋯）與「它可能開出什麼未來選項」；另有身份彈性改寫——把「我就是做這個的」改寫成保留彈性的版本。
- **回顧**：資產數、分散度、最新彈性分、探索次數，以及「探索開出的未來選項」清單。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`strategic_optionality_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式

直接用瀏覽器開啟 `index.html`，或加到手機主畫面當 App 用。
