# 複利人生系統 · Compounding Life System

最強的成長，不是偶爾爆發，而是讓生活本身變成會持續複利的系統。複利概念源自金融，被延伸到個人成長：真正拉開差距的，往往不是一次巨大努力，而是那些會持續累積、彼此加成的小系統。

**線上使用：https://iamkeith001.github.io/productivity-systems/compounding-life/**

## 核心概念

一般努力是「做完就結束」，複利系統是「做完之後還會留下東西」。讀一本書只是讀完，價值停在當下；整理成筆記、提煉成觀點、用在決策中，這本書就進入了複利系統。

## 功能

- **沉澱**：複利存款——每完成一件事記下「留下了什麼給未來」，依六種複利維度分類（輸入／知識／輸出／能力／關係／系統），頂部六格即時顯示各維度累積。
- **轉化**：輸出轉化鏈——「一次輸出，不只服務一次」。替一份成果列出轉化清單（長文→簡報→課程→圖卡），完成一項勾一項，全部完成時顯示「一次思考被用了 N 次」。
- **健檢**：複利系統五特徵自評（輸入被篩選／知識被整理／輸出被重用／經驗被回收／系統自我修正），複利指數長條圖與歷次對比。
- **回顧**：累計存款、連續沉澱天數、轉化完成進度、最新複利指數。

## 與「複利行動飛輪」的差別

- **複利行動飛輪**（`compound-flywheel/`）：追蹤單一飛輪習慣的每日推動與動能。
- **複利人生系統**（本工具）：管理整個生活的複利結構——每件事留下什麼、成果如何被重複轉化、系統五特徵是否健全。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`compounding_life_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式

直接用瀏覽器開啟 `index.html`，或加到手機主畫面當 App 用。
