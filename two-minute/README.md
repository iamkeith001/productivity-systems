# 兩分鐘啟動法則 · Two-Minute Rule Plus

用兩分鐘啟動行動，引爆動能，克服拖延。核心不是「只做兩分鐘」，而是先讓行動開始、再讓動能接手。概念源自 David Allen 的 GTD 兩分鐘法則與 James Clear《原子習慣》的延伸。

**立即使用：https://iamkeith001.github.io/productivity-systems/two-minute/**

## 功能

- **入口**：把原始任務改寫成「兩分鐘入口」（可以馬上開始的第一步），分為完成型（兩分鐘做掉）與啟動型（做不完但可開始），可加固定觸發點；內建兩分鐘陷阱檢查。
- **啟動**：兩分鐘啟動計時器。時間到後不強迫繼續，只問「願不願意再做五分鐘？」——有退出權才更願意開始。動能接手就一段段延伸，每段記錄。
- **戰績**：統計總啟動次數、延伸率（動能接手比例）、完成型清掉數，並列出每次啟動紀錄。
- **學習**：三層結構、改寫範例、兩分鐘後選擇權、避免陷阱、應用案例。

## 與行動觸發的關係

把兩分鐘行動接在固定情境後面，就與「行動觸發」（action-trigger）的習慣堆疊串成更穩定的行動系統。
