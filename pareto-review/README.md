# 80/20 檢視法 · 80/20 Review Method

定期檢視產出與投入比例，聚焦真正重要的事。源自帕雷托法則（Pareto / 80-20）：少數關鍵行動造成多數結果。

**立即使用：https://iamkeith001.github.io/productivity-systems/pareto-review/**

## 功能

- **盤點**：列出佔用時間的活動，替每件標上投入（時間／精力 1–5）與產出／價值（1–5），自動歸入象限。
- **檢視**：投入 × 產出 2×2 象限視覺化（高槓桿·放大／核心·必要／消耗·該調整／瑣事·批次）；帕雷托計算——「產出最高的 N 件事撐起 80% 價值只花 X% 投入」「投入最多的 M 件事吃掉 80% 時間只換 Y% 價值」；替每件活動指派調整行動（保留加強／減少／委派／自動化／建 SOP／刪除）。
- **節奏**：每週三個固定問題（最有價值的 20%？最消耗的 80%？下週移向哪件高價值任務？），存成檢視紀錄追蹤重心移動。
- **學習**：五步驟、應用案例與專業解讀。

## 與其他工具的關係

80/20 檢視告訴你「該把時間移向哪些事」；「能量區塊」決定放哪個精神時段；「時間分塊」把它排進行事曆。檢視 → 排程一氣呵成。
