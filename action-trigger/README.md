# 行動觸發設計 · Action Trigger Design

把《行動觸發設計（Action Trigger Design）》變成日常好習慣啟動的設計與調校工具。

**立即使用：https://iamkeith001.github.io/productivity-systems/action-trigger/**

單一 HTML 檔、零外部相依、離線可用、資料只存在你自己的裝置（瀏覽器 localStorage）。

---

## 核心理念

> 不要只問「我要不要做」，而是設計「什麼情境一出現，我就自然開始做」。

行動觸發設計源自 BJ Fogg 行動模型（Behavior = Motivation + Ability + Prompt）。好的提示（Trigger）必須明確、貼近、低摩擦。

本工具協助你將模糊的好習慣目標，組裝成一個自動啟動的 If-Then 語句：

```text
明確目標行為 → 找出自然觸發點 (Anchor) → 設計 < 2 分鐘的最小啟動動作 → 設計環境提示可視化 → 定期回顧與優化
```

## 四大分頁

| 分頁 | 功能說明 |
|---|---|
| **設計** | 4 步驟精靈引導設計觸發器。支援常見觸發點範本點擊帶入，自動拼接為標準 If-Then 句型。 |
| **觸發器** | 列出所有運作中的觸發器與環境可見提示，顯示下次需要回顧的進度狀態。 |
| **回顧** | 依據習慣科學 3 大問（是否出現？是否注意到？是否立刻開始？），客觀分析執行瓶頸。**別責怪意志力**，而是科學調整觸發點或降低難度。 |
| **統計** | 視覺化呈現習慣啟動成功率，以及習慣優化方案的佔比統計。 |
