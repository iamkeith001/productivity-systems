# 時間分塊法 · Time Blocking

透過時間分塊安排，減少多工干擾，讓每日計畫更可執行。核心來自時間管理與 Cal Newport《Deep Work》——用行事曆保護不被打斷的專注時段。

**立即使用：https://iamkeith001.github.io/productivity-systems/time-blocking/**

## 功能

- **任務**：列出今天所有佔用注意力的事，分成三類——深度工作（高專注）、淺層工作（可批次）、固定事件（已排定）。
- **時間表**：把任務排進時間區塊（開始/結束時間），視覺化時間軸依時間排序；自動偵測區塊重疊（紅色警告）與緩衝間隔，頂部顯示深度/淺層/固定時數與總緩衝。
- **檢視**：注意力分配長條圖、已排時數與緩衝總計；計畫健檢——重疊、連續無緩衝、深度工作是否在上午、淺層工作是否該批次。
- **學習**：一天的樣子、五步驟、應用案例與專業解讀。

## 與其他工具的關係

「能量區塊」幫你決定哪個時段精神最好、該放深度工作；「認知負荷」提醒別把過重任務塞進零碎時間。三者互補。
