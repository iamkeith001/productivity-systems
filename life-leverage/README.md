# 人生槓桿設計 · Life Leverage Design

真正的成長，不是什麼都親自做，而是讓少數關鍵行動放大更多結果。槓桿概念源自物理學，被引入管理、投資與個人成長領域：高效不是把時間塞滿，而是找出哪些行動能產生更大的後續效果。

**線上使用：https://iamkeith001.github.io/productivity-systems/life-leverage/**

## 核心概念

低槓桿行動是「做一次，只產生一次結果」；高槓桿行動是「做一次，未來能被重複使用、複製、放大或自動運作」。寫一封臨時回覆是低槓桿，建立一套回覆模板是高槓桿——從「時間勞動者」變成「系統設計者」。

## 功能

- **支點**：盤點高槓桿領域（能力提升／流程標準化／長期作品／關係建立），標上六種放大器（時間／知識／技術／關係／資本／品牌），頂部放大器覆蓋格即時顯示缺口。
- **資產**：可複製系統資產庫（模板、SOP、checklist、prompt 庫、知識庫、自動化流程、決策準則）。核心機制：每次重複使用按「重用 +1」，累計重用與**平均槓桿倍數**即時計算，滿 5 次標「高槓桿資產」。
- **淘汰**：低槓桿行動清單，指派處置方式（停止／委派／自動化／模板化／批次化），打勾標記已處置。
- **回顧**：支點數、資產數、累計重用、已處置比率＋「重用最多的系統資產」槓桿排行。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`life_leverage_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式

直接用瀏覽器開啟 `index.html`，或加到手機主畫面當 App 用。
