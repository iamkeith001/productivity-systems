# 限制條件重構 · Constraint Reframing

高手不是沒有限制，而是能把限制重新設計成創造力來源。概念源自 Goldratt 的限制理論（Theory of Constraints），延伸到創意設計、知識工作與個人成長。

**線上使用：https://iamkeith001.github.io/productivity-systems/constraint-reframing/**

## 核心概念

低階思維：限制＝阻礙。高階思維：限制＝設計條件。面對限制時不只是抗拒它，而是重新定義它的意義與可能性——把「我做不到」轉化成「我如何在這個條件下創造最大價值」。

## 功能

- **辨識**：記錄限制並診斷「真正卡住系統的是什麼」——很多時候不是時間或資源，而是決策混亂、注意力分散、方向不清。
- **重構**：四個重構問題（逼我聚焦什麼？迫使我簡化什麼？創造什麼不同做法？形成什麼獨特優勢？），把受害模式切換成設計模式。
- **行動**：把設計條件轉成具體的限制導向行動（可打勾完成）；另可建立「限制模擬」——主動加限制（限時、限篇幅、限工具）並累計實行次數。
- **回顧**：限制數、已重構數、行動完成率、模擬實行次數，以及「限制替你形成的獨特優勢」清單。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`constraint_reframing_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式

直接用瀏覽器開啟 `index.html`，或加到手機主畫面當 App 用。
