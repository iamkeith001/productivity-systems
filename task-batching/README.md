# 任務批次化 · Task Batching

把相似類型、相似工具、相似思考模式的任務集中在同一段時間處理，減少情境切換成本，提升專注力與整體產出效率。

**線上使用：https://iamkeith001.github.io/productivity-systems/task-batching/**

## 核心概念

任務批次化不是多工，而是反多工。多工是同時切換多件事；任務批次化是一次只處理同一類事，讓大腦保持在同一種「工作模式」中，減少反覆啟動、停止、重新進入狀態的成本。

## 功能

- **收集箱**：新任務出現時先暫存並標上認知頻道（溝通／創作／分析／行政／學習），不立刻切換去做。
- **批次時段**：替每種頻道預約固定時間盒，附邊界規則（例如 Email 只在固定時段開），內建批次大小建議。
- **執行批次**：選一個認知頻道，勾選這一批要處理的任務，一次結算記成一個批次。
- **回顧**：累計批次、清空任務數、平均每批、最常批次頻道與逐批紀錄。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`task_batching_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式

直接用瀏覽器開啟 `index.html`，或加到手機主畫面當 App 用。
