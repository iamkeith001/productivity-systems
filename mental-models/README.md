# 心智模型清單 · Mental Models Library

把《心智模型清單（Mental Models Library）》變成日常分析複雜問題、進行多元維度決策與防止認知盲點的思考工具箱。

**立即使用：https://iamkeith001.github.io/productivity-systems/mental-models/**

單一 HTML 檔、零外部相依、離線可用、資料只存在你自己的裝置（瀏覽器 localStorage）。

---

## 核心理念

> 擁有多元心智模型，就像擁有多副眼鏡；它讓你看得更遠、更準、更全面。

人的直覺與思考極易受限於單一角度。心智模型不是知識本身，而是使用知識的方式。本工具提供一個結構化的思考工具箱，並內置 15 個常見的核心心智模型：

- **決策模型**：機會成本、二階思考、期望值。
- **系統模型**：系統思維、瓶頸理論、複利效應。
- **學習模型**：費曼學習法、刻意練習、輸出導向學習。
- **行為模型**：習慣迴圈、損失規避、社會認同。
- **風險模型**：安全邊際、冗餘設計、預演失敗。

## 四大分頁

| 分頁 | 功能說明 |
|---|---|
| **模型庫** | 瀏覽與搜尋預置及自訂的模型。點擊卡片可展開查看「適用情境」、「提醒問題」與「常見誤用」。可在此標記核心模型「★」。 |
| **記模型** | 新增或編輯你的自訂心智模型，充實個人思考工具箱。 |
| **思考沙盒** | 互動式決策分析表。輸入當下面臨的分析主題，勾選想要套用的心智模型，系統將動態產出問答表單，引導你深入回答每個模型的提醒問題，並儲存為日誌。 |
| **統計日誌** | 查看歷史思考日誌，分析自己最常用的思維模型，並進行備份與還原。 |
