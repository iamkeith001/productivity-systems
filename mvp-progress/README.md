# 最小可行進展法 · Minimum Viable Progress

透過每天一點點進展，累積長期成果並保持動力。概念延伸自創業的 MVP（最小可行產品）：不等狀態完美才開始，每天完成一個足以推動事情前進的最小行動。

**立即使用：https://iamkeith001.github.io/productivity-systems/mvp-progress/**

## 功能

- **目標**：替每個長期目標定義「最小完成單位」（每天 N 個單位，如 100 字、1 篇、5 個單字），可設總目標量顯示完成進度；含即時公式預覽與範本。
- **推進**：每天記錄微進展——數量、完成內容、下一步/問題。達最小單位提示「達標」，未達但有推進也算前進；完成度圓環顯示今天動了幾個目標。
- **進度**：近 14 天推進格（實心＝達最小、淺色＝有推進）、目前/最長連續天數、累積量與總目標進度；可就地放大或降回最小單位（記錄調整歷史）。
- **學習**：最小單位怎麼定、五方法、應用案例與專業解讀。

## 與其他工具的關係

「兩分鐘啟動」幫你跨過開始門檻；「微習慣堆疊」把每日推進接在既有習慣後面；本工具追蹤長期累積與連續性。

## 特性

- 單檔 HTML，純前端、離線可用
- 資料只存本機 localStorage，可下載／讀取 JSON 備份
- 深淺色主題、支援加入手機主畫面
