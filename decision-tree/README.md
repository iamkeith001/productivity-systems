# 決策樹思維 · Decision Tree Thinking

用結構化方法分析選項，做出更明智的決策。把複雜問題拆解成「問題、條件、選項、結果」，讓判斷更理性、可討論、可修正。源自統計學、運籌學與決策分析。

**立即使用：https://iamkeith001.github.io/productivity-systems/decision-tree/**

## 功能

- **分析（五步精靈）**：① 定義真正的問題（從「選 A 還是 B」重構成「真正要解決什麼」）② 找關鍵判斷條件並加權（時間/金錢/風險/長期價值/可逆性…，1–5 權重）③ 展開多個選項（含中間路徑範本）④ 加權評分矩陣＋每選項最好/最壞結果 ⑤ 算出建議路徑、選定決定、設檢查點。
- **決策庫**：存下的決策，顯示決定路徑、是否採用系統建議、各選項評分與最好/最壞結果；統計決策數、採用建議數、待辦檢查點。
- **檢查點**：跨決策的所有檢查點清單，回頭勾選追蹤——讓決策保留彈性而非僵硬走到底。
- **學習**：四元素、五步驟、應用案例與專業解讀。

## 與其他工具的關係

「決策樹」幫你當下做結構化選擇；「決策日誌」記錄當下假設與信心供事後複盤；「心智模型」提供多元思維模板。三者互補。
