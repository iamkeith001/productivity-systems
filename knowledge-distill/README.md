# 知識蒸餾法 · Knowledge Distillation

把 Hinton 等人的知識蒸餾精神應用到知識管理：從大量資料中萃取核心洞見，建立可複用的知識精華。

**立即使用：https://iamkeith001.github.io/productivity-systems/knowledge-distill/**

## 功能

- **蒸餾工作台**：三步驟精靈——①明確蒸餾目的（寫報告／做決策／教學／建立 SOP／研究假設）②倒入素材並歸類為事實／觀點／原則／行動 ③萃取一句話洞見＋三點摘要＋下一步行動，存成知識卡片。
- **卡片庫**：搜尋與依目的篩選；每張卡可展開原始素材、一鍵複製純文字精華（【主題】【一句話洞見】【三點摘要】【下一步行動】）。
- **回流檢核**：實際應用後記錄「反覆有效／部分有效／無法支持行動」。有效 2 次以上自動升級為「標準」；失效卡片標記待重新蒸餾，可帶回工作台重煉。
- **學習**：理論背景、蒸餾與摘要的差別、五步驟、應用案例與專業解讀。

## 特性

- 單檔 HTML，純前端、離線可用
- 資料只存本機 localStorage，可下載／讀取 JSON 備份
- 深淺色主題、支援加入手機主畫面
