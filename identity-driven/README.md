# 身份驅動系統 · Identity-Based Systems

長期改變最穩的方式，不是逼自己做事，而是成為會自然做那些事的人。概念源自自我認同理論與行為科學：長期行為往往不是由目標決定，而是由自我認同決定。

**線上使用：https://iamkeith001.github.io/productivity-systems/identity-driven/**

## 核心概念

「我做這件事，不只是為了達成目標，而是因為這就是我是誰。」目標有完成的一天，身份則會持續影響未來選擇——目標驅動容易短期衝刺，身份驅動更容易長期維持。

## 功能

- **身份**：把目標句改寫成身份句（「我要讀 50 本書」→「我是持續學習的人」），依領域分類；卡片顯示累積票數，滿 10 票標「身份成形中」。
- **投票**：身份證據紀錄——每個小行動都是一張選票。統計今日投票、累計投票與連續投票天數。
- **設計**：環境設計（讓環境每天強化你要的身份）＋更新舊身份（把「我是拖延的人」等限制性標籤改寫成新身份）。
- **回顧**：身份數、累計選票、連續天數、環境＋標籤數；正向循環示意（身份→習慣→結果→強化身份）；各身份的「身份強度」票數長條。

## 特色

- 純前端單一 HTML 檔案，可離線使用
- 資料只存本機 `localStorage`（key：`identity_driven_v1`）
- 支援 JSON 備份下載／還原
- 深淺色主題（跟隨系統＋手動切換）
- 行動裝置友善（44px 觸控熱區、safe-area）

## 使用方式

直接用瀏覽器開啟 `index.html`，或加到手機主畫面當 App 用。
